Saya membuat desain Aplikasi Pemesanan makanan dan minuman.
Aplikasi pemesanan makanan adalah program sederhana berbasis desktop yang digunakan untuk mencatat dan mengelola pesanan makanan dan minuman pada sebuah tempat makanan/restoran.
Aplikasi ini memungkinkan penggguna memasukkan data pemesan, memilih menu, serta menampilkan struk pesanan secara otomatis.
Fitur Utama : 
- Input nomor meja
- Input nama pemesan
- Pilihan kategori (misalnya: makanan/minuman)
- Daftar menu makanan
- Menampilkan harga menu
- Generate struk pesanan otomatis
- Tombol “Pesan” untuk memproses pesanan

Tampilan Aplikasi :
Pada tampilan utama terdapat beberapa bagian:
- No. Meja → untuk mengisi nomor meja pelanggan
- Pemesan → untuk mengisi nama pelanggan
- Kategori → dropdown untuk memilih jenis menu
- Menu → daftar pilihan makanan
- Harga → menampilkan harga dari menu yang dipilih
- Struk Pesanan → menampilkan ringkasan pesanan
- Tombol PESAN → untuk memproses dan menampilkan struk

  Cara Menjalankan Aplikasi
  1. Buka project di NetBeans 
  2. Jalankan file utama 
  3. Isi data:
     - Nomor meja
     - Nama pemesan
  4. Pilih kategori dan menu
  5. Klik tombol PESAN
  6. Struk akan tampil di bagian kanan

Cara Kerja Singkat
1. User menginput data (meja & nama)
2. User memilih menu
3. Sistem mengambil harga sesuai menu
4. Saat tombol Pesan ditekan:
   - Data ditampilkan ke struk
   - Format struk dibuat otomatis
